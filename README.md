# HW5 WebSocket Protobuf Chat

Клиент-серверное приложение чата на WebSocket и бизнес-протоколе на Protobuf для задания HW5 по компьютерным сетям.

Сервер реализован на Java 17 с использованием Netty.  
Клиент реализован на чистом JavaScript без frontend framework.  
Клиент и сервер обмениваются сообщениями через WebSocket Binary Mode.  
Бизнес-протокол описан в `src/main/proto/chat.proto` и реализован на Protocol Buffers.

## Возможности

- подключение пользователя по имени;
- проверка уникальности имени;
- публичные сообщения;
- приватные сообщения через формат `@name text`;
- изображения JPEG/PNG до 1 МБ;
- иконка пользователя при подключении;
- история последних 50 сообщений;
- история последних 50 изображений;
- последнее видимое изображение при подключении;
- обработка ошибок через protobuf-ответы;
- повторное подключение клиента через 10 секунд.

## Структура проекта

```text
src/main/java/ru/hse/hw5/      # серверная часть
src/main/proto/chat.proto      # protobuf-протокол
src/main/resources/client/     # HTML/JS/CSS клиент
src/test/java/ru/hse/hw5/      # тесты
````

## Запуск сервера

Из корня проекта:

```bash
mvn exec:java
```

После запуска сервер доступен по адресу:

```text
ws://localhost:8080/chat
```

## Запуск клиента

Клиент находится в папке:

```text
src/main/resources/client
```

Запуск через простой static server:

```bash
npx serve src/main/resources/client
```

После этого открыть адрес из консоли:

```text
http://localhost:3000
```

## Использование

1. Запустить сервер.
2. Запустить клиент.
3. Открыть клиент в браузере.
4. Ввести имя пользователя.
5. Нажать `Присоединиться`.
6. Отправлять публичные сообщения или приватные сообщения в формате:

```text
@Bob hello
```
